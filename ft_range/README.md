課題名: ft_range
必要なファイル: ft_range.c
使用可能な関数: malloc

---

以下の関数を作成してください：

int \*ft_range(int start, int end);

この関数は、malloc()を使用して整数の配列を確保し、start で始まり end で終わる連続した値で
配列を埋め（start と end を含む！）、配列の最初の値へのポインタを返す必要があります。

例：

- (1, 3)の場合、1, 2, 3 を含む配列を返します。
- (-1, 2)の場合、-1, 0, 1, 2 を含む配列を返します。
- (0, 0)の場合、0 を含む配列を返します。
- (0, -3)の場合、0, -1, -2, -3 を含む配列を返します。
